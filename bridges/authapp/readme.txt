Для обработки поля ImageField необходимо установить графическую библиотеку Pillow, используя команду:
    pip install pillow

Ограничения прав доступа в шаблоне...

По умолчанию все новые зарегистрированные пользователи не активированы. Менеджер компании по телефону связывается с
пользователем, подтверждает его личность и активирует аккаунт (галочка в админке).
